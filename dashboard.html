What This Prototype Does

This system uses rainfall, humidity, drainage, land surface, soil, slope, groundwater, construction, and previous rainfall saturation data to estimate drying behavior after rainfall.

Recommended Municipal Use

Authorities can use the priority zones to schedule drain cleaning, improve water diversion, monitor low-lying areas, and plan permeable infrastructure.
